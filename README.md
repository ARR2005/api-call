# api-call

This template should help get you started developing with Vue 3 in Vite.

#Objective 
 Learn JavaScript by building a project making API calls and integrating basic styling. You will learn fundamental JavaScript concepts: variables, functions, loops, and conditionals.

 #Criterias to be followed 
 ###Auth: Determine the type of authentication required (e.g., No, API Key, OAuth). Those who want to challenge themselves may choose OAuth.
 ###HTTPS: This shows whether the API supports HTTPS (secure HTTP). This is important for ensuring data security during transmission.
 ###CORS: Cross-Origin Resource Sharing (CORS) indicates whether the API allows requests from different origins (like your web application).

 #Approach 
 -VUEJS will be utilized as the front-end framework.
 -API calls will be made using Axios.

 "The project will be built using a simple API call to a public API to fetch data and display
 but able to call more than one api  and display the data in a specific Box that will open when a specific categories is choosen

 I planned to used Axios a javascript library used for making HTTP requests from the browser. For Easier approach for this problem. "
